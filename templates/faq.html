Frequently Asked Questions

What is Vasco de Gapi? What does it explore?

Vaso de Gapi explores the newly released Google Analytics API. You can emulate any type of data pull available through the API. Vasco de Gapi handles it for you. For every Google Analytics API pull, you can also download the code used to make that specific API call, to ease automation of API data pulls.

Do I need Google Analytics to use Vasco de Gapi?

Yes! Get it at the Google Analytics site.

I am not getting back any data. What is wrong?

I likely means that the selected data slice has a limited amount of data available. You may have selected a very restrictive filter, or picked the wrong site in Google Analytics. The date ranges could be from a time before the site was set up. The best thing to do could be to attempt to replicate the profile/filter/selection in the actual Google Analytics website.

How many observations can Vasco de Gapi pull from the API?

Currently, Vasco de Gapi is limited to pulling 10,000 observations. If you need to pull more data, you can convince your code monkeys to download the automatically generated code, and scale up the limitations. You may also leave a comment {{here}}, and if this is a highly-desired feature, we may consider increasing the limit.

Can I sort by more than one dimension or metric?

Not yet. Leave a comment {{here}}, and we just might add in that feature.
